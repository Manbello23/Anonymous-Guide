To add Day N:
1. Open script.js
2. In daysData array, create a new object:
   {
     day: N,
     title: "Short Title",
     ayahArabic: "Arabic ayah here (optional)",
     ayahTrans: "Translation (optional)",
     text: `Full text here (use backticks for multiline)`
   }
3. Save and push to GitHub (or re-upload). The site will show the new day in the grid.
4. Users unlock days locally (Unlock Next button) after 24 hours.
